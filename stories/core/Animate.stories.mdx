# 动画上下文

动画上下文是专为组件机制设计的一套动画控制逻辑。
组件提供者使用由此提供的`AnimateBase`包装设计好的组件，
使用者则可以通过`AnimateContext`为此类组件设置是否展示的状态，从而做到更顺滑、流畅的界面切换效果。

在使用动画上下文进行状态控制之前，需要先定义一个以`AnimateBase`包装的组件。

<Canvas>
  <Story name="AnimateContext 使用示例" args={{}}>
    <AnimateContext show>
      <AnimateBase
        styles={{
          hasEnter: 'button-enter',
          onEntering: 'button-entering',
          hasLeave: 'button-leave',
          onLeaving: 'button-leaving',
        }}
      >
        <button className="button">测试</button>
      </AnimateBase>
    </AnimateContext>
  </Story>
</Canvas>
